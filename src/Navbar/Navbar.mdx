## Navbar with menu

  This navbar is specific for showing menu items. Only this Navbar have the colapse
  comportament is controlled by `mode` prop, passing `'allways'` the hamburger button
  is allways shown, but passing `'mobile'` it's shown just on mobile devices.

  >To test collapse behavior, you need to use browser resizing

<Playground>
  <NavbarMenu
    mode='mobile'
    menuItems={

      </>
